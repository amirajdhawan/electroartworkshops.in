PHP & MySql

PHP (Hypertext Preprocessor) is a widely-used open source general-purpose server side scripting language. It is widely used with MySql as a backend, which is an open source Relational Database Management System and popular choice of database for use in web application. This includes HTML embedded with PHP forming the front end of web application, through which back end mysql database can be accessed, involves data insertion, updatation, deletion etc.



Arduino

Arduino is an open-source electronics prototyping platform based on flexible, easy-to-use hardware and software. Arduino can sense the environment by receiving input from a variety of sensors and can affect its surroundings by controlling lights, motors, and other actuators. In this demo workshop, a line follower robot will be made using arduino.
