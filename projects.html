Remote Controlled Car Robot

Remote controlled car comes with two wheels connected to motors and a third trolley wheel. The remote contains 2 buttons which can be used to control the car. The entire robot is hand made using basic electronic components and cardboards without any readymade parts! Provides a deep understanding of the components since only basic parts are used.
